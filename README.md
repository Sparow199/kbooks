# KBooks

### Description

Book management application that uses Kotlin and Spring Boot 2 framework.

### Libraries

- Spring 5
- Jersey
- Hibernate (with JPA)
- H2
- JUnit (with kotlin-test)
- Mockito (with mockito-kotlin)

### Run the app

Execute this command in a terminal:

```bash
./gradlew bootRun
```

Open `http://localhost:8080/` in your browser
